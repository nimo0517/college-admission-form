Q.What is SSH ?
A.Secure Shell Protocol-is a network communication protocol that enables two computers to communicate 
(c.f http or hypertext transfer protocol, which is the protocol used to transfer hypertext such as web pages) 
and share data.

Q.Why do we need SSH?
A.SSH encrypts and authenticates all connections. SSH provides IT and information security (infosec) professionals
with a secure mechanism to manage SSH clients remotely. Rather than requiring password authentication to initialize
a connection between an SSH client and server, SSH authenticates the devices themselves.

Q.How do I use SSH?
